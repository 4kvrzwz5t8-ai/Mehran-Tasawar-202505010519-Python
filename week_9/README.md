# Week 9 - Helpdesk Ticket

This folder contains a simple helpdesk ticket registration system.

Files:
- `main.py` - Runs the ticket flow.
- `ticket.py` - Collects ticket details from user input.
- `display.py` - Displays the ticket in a formatted way.

Usage:

```bash
python3 week_9/main.py
```

Or run non-interactively:

```bash
printf "izzad\n12345678910\nblue screen pc\nlab 101 level 1\n1\n" | python3 week_1/main.py
```
